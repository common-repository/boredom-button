== Description ==

Places the Boredom Button on your wordpress blog, giving your users interesting content from the net.
== Installation ==

1. Download boredom-button and place in wp-content/plugins
2. Log into your WordPress admin panel
3. Go to Plugins and "Activate" the plugin
4. Go to Appearance->Widgets to add it to your sidebar
5. If you haven't set up a sidebar before, you'll have to add your old widgets back too
